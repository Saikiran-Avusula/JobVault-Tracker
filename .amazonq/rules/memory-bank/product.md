# Product Overview

## Project Purpose
JobVault Tracker is a personal job application management system designed to help job seekers organize, track, and manage their job applications throughout the entire job search lifecycle.

## Value Proposition
- Centralized tracking of all job applications in one place
- Status management from application submission through final decision
- Secure authentication with Google OAuth via Supabase
- Persistent storage with soft-delete functionality (trash/restore)
- Dark/light theme support for comfortable extended use
- Responsive design for desktop and mobile access

## Key Features

### Core Capabilities
- **Application Management**: Create, read, update, and delete job applications
- **Status Tracking**: Monitor application progress through various stages (Applied, Interview, Offer, Rejected)
- **Detailed Records**: Store company name, position, salary range, location, job description, and notes
- **Soft Delete**: Move applications to trash instead of permanent deletion, with restore capability
- **Search & Filter**: Quickly find applications by company, position, or status
- **Authentication**: Secure Google OAuth login via Supabase

### User Experience
- Clean, modern UI with Tailwind CSS
- Toast notifications for user feedback
- Modal-based workflows for creating and editing applications
- Confirmation dialogs for destructive actions
- Error boundaries for graceful error handling
- Theme persistence across sessions

## Target Users
- **Primary**: Individual job seekers actively applying to multiple positions
- **Use Case**: Personal project for managing job search workflow
- **Skill Level**: End users with basic web application familiarity

## Project Category
**PERSONAL PROJECT** - Prioritizes speed, simplicity, and single-user maintainability with minimal dependencies and easy local development.
